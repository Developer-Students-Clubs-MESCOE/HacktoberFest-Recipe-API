Node.js Recipe API

Write a Node.js API using Express to work with recipes in MongoDB

NOTE - This repository is labeled hacktoberfest or will be accepted for Hacktoberfest only till 30th October 2020 11:59 PM IST

How to get started?

  1. Create a Github account if you don't already have one.
  2. Register yourself at Hactoberfest 2020 using your Github account if you haven't already.
  3. Fork this repository and clone in onto your machine.
  4. Check this out Installing MongoDB Community Edition to install MongoDB onto your system. Use MongoDB Compass to view your data, if you're unfamiliar with the mongo commands.
  5. Run npm install or yarn install to install the required dependencies.
  6. Run npm start or yarn start
  7. Check out the Projects tab for features to be added.
